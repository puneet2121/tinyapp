## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session
- mocha and chai 
- nodemon

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.
- Type in npm start to start your server.
